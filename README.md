# MealMap 🍽️

A mobile app that helps users choose recipes based on store selection, price per serving, and shopping efficiency.

## Features (Planned)
- Recipe browsing with ratings and reviews
- Store-based grocery list generation
- Price per serving calculation
- Personalized recommendations
- Shopping time estimates

## Tech Stack
- React Native (Expo)
- TypeScript
- Node.js (planned)
- PostgreSQL (planned)
